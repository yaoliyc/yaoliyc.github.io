盒模型-宽度

  1. 没有（就是没有设置width的）宽度的元素始终会扩展到填满其父元素的宽度位置。添加水平边框、内边距和外边距，会导致内容宽度减少，减少量等于水平边框、内边距和外边距的和。
  2. 为设定了宽度的盒子添加边框、内边距和外边距，会导致盒子扩展的更宽。实际上，盒子的width属性设定的只是盒子内容区的宽度，而非盒子要占据的水平宽度。

浮动与清除

围住浮动元素的三种方法

  1. 为父元素添加overflow:hidden
  2. 同时浮动父元素
  3. 添加非浮动的清除元素，可以直接在标记中加，也可以通过给父元素添加clearfix类来加

定位

  1. 默认静态定位，默认文档流。
  2. 相对定位，相对于原来在文档流中的位置（或者默认位置），默认的空间仍然被占据，默认文档流
  3. 绝对定位，默认相对于body定位，脱离默认文档流。
  4. 固定定位，相对于视口定位，不随页面滚动而移动，脱离默认文档流。

定位上下文:绝对定位元素默认的定位上下文是body，绝对定位元素的任何postion为relative的祖先元素。

显示属性

  • 块级元素，比如段落、标题、列表等，在浏览器中上下堆叠显示。
  • 行内元素，比如a、span和img，在浏览器中左右并排显示，只有前一行没有空间才会显示到下一行。
