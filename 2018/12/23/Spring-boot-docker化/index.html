环境准备

  • JDK 1.8或以上
  • Maven 3.2+
  • IntelliJ IDEA
    如果使用的不是Linux机器，需要安装虚拟机，VirtualBox或MAC上的boot2docker。

Spring Boot Maven plugin 提供许多方便的功能

  • 收集类路径上的所有jar并构建一个可运行的jar包，这使得执行和传输更加方便。

  • 搜索public static void main（）方法以标记为可运行的类。

容器化

Docker使用Dockerfile来定义一个image。因此需要为Sring Boot项目创建一个Dockerfile。

该Dockerfile有一个DEPENDENCY参数指向jar包的解压位置。正确情况下，它包含BOOT-INF/lib目录，该目录下有所有依赖jar包，和BOOT-INF/classes目录，此目录下包含应用的classes编译文件。
